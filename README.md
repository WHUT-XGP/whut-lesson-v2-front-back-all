#### 基于Python后端Flask和前端Uniapp开发的WHUT课表

前端：

WHUTLessonV2文件夹，基于uniapp开发

后端：

LessonV2文件夹，采用Flask开发，数据爬取自武汉理工大学教务官网

下载release包下的apk文件安装使用

后端启动 python app.py

前端启动 在Hbuilder下导入uniapp项目
